 Used to decorate xUnit.net test classes and collections to indicate a test which has per-test-class fixture data. To gain access to the fixture data from inside the test, a constructor argument should be added to the test class which exactly matches the TFixture . enum Xunit.CollectionBehavior
strong

Defines the built-in behavior types for collections in xUnit.net.

Enumerator
CollectionPerAssembly 

By default, generates a collection per assembly, and any test classes that are not decorated with CollectionAttribute will be placed into the assembly-level collection.

CollectionPerClass 

By default, generates a collection per test class for any test classes that are not decorated with CollectionAttribute.
